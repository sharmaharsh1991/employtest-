

# Employee crud using latest laravel with Vue3.

## Clone the project

```bash
$ git clone https://github.com/detailswes/employtest.git
$ cd employtest
```
## Install npm dependencies

```bash
$ npm install
```

## Migration

```bash
$ php artisan migrate
```


## Vite Dev Server Start

```bash
$ npm run build
```

## Start the Local server

```bash
$ php artisan serve
```
